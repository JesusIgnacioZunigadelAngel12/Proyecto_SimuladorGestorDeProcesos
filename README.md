# Simulador de Gestor de Procesos - Modo Batch Automático

Simulador educativo desarrollado en Python con GUI en Tkinter que demuestra los algoritmos de planificación FCFS y SJF, además de comunicación entre procesos (IPC).

Para iniciar la simulación, ejecuta el siguiente comando en la terminal:

